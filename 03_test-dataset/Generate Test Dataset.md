# Generate Test Dataset

### **Description**

> Create a comprehensive test dataset to support QA testing of FlowCRM core workflows.

### **Acceptance Criteria:**

- 100+ client records with varied data:
 - 30% status "New"
- 50% status "In Progress"
- 20% status "Done"

### **Include 20+ edge cases:**
- Very long names (100+ characters)
- Special characters in fields (émojis, accents)
- Various phone formats (+380..., (044)..., etc.)
- Empty optional fields
- Boundary values (very high/low deal values)
-  50+ notes distributed across clients
-   Data ready for import (CSV format)

### **Deliverable:** 
>  CSV export test data
